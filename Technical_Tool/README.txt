============================================================
   NetTrace Automated Forensic Analyzer
   ITT593 Project Submission
============================================================

[1] SYSTEM REQUIREMENTS
------------------------------------------------------------
- Operating System: Windows 10/11
- Python Version: Python 3.8 or higher
- Wireshark: Must be installed (specifically the 'tshark' component)

[2] INSTALLATION INSTRUCTIONS
------------------------------------------------------------
1. Install Python from https://www.python.org/ (Check "Add to PATH").
2. Install Wireshark from https://www.wireshark.org/.
3. Open Command Prompt and run:
   pip install -r requirements.txt

[3] HOW TO RUN
------------------------------------------------------------
Double-click 'launcher.bat' to start the tool in forensic mode.
